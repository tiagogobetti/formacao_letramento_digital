# Formação Letramento Digital Docente

> Aprender. Experimentar. Transformar.

Plataforma desenvolvida para apoiar o terceiro encontro da Formação Letramento Digital Docente, com foco na utilização do Google Workspace for Education no contexto educacional.

---

## Objetivos

Proporcionar aos profissionais da educação oportunidades para conhecer, explorar e aplicar os recursos do Google Workspace em práticas pedagógicas, administrativas e de gestão escolar.

---

## Público-alvo

- Professores
- Equipes Gestoras
- Supervisores
- Orientadores
- Assistentes Técnico-Pedagógicos
- Assistentes de Educação
- Administradores Escolares

---

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
- Google Apps Script
- Google Sheets
- GitHub Pages
